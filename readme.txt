== Description ==

Visual Feedback enables you to provide feedback on your WordPress site by simply clicking and commenting. Comments and other related information (like the code of the commented-on element) are sent to your Timeline.io account for easy review.



== Installation ==

This section describes how to install the plugin and get it working.

1. Log in to WordPress as an administrator.
2. Install the plugin via the Plugins screen.
3. Activate the plugin via the Plugins screen.
4. Configure the plugin by clicking "Visual Feedback" in your admin menu.

Alternatively, you may directly upload the "visual-feedback" plugin folder to the "/wp-content/plugins" folder of your WordPress site on your web server.



== Frequently Asked Questions ==

= Do I need a Timeline.io account? =

Yes, but Timeline.io accounts are **free**!
Need an account? Sign up here: [https://www.timeline.io/](https://www.timeline.io/)

= How can I review feedback? =

Option A: With feedback enabled (see your plugin settings), browse your WordPress site as you normally would and any elements with comments on them will be highlighted and a comment count will be shown. Click on any of these elements to view all comments on that element.

Option B: Review all comments and other related information (like the code of the commented-on element) by clicking "View Feedback" from the plugin's settings page.
